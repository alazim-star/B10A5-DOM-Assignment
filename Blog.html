What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that browsers provide to interact with and manipulate the structure of web documents.and It represents the structure of a document as a tree of objects, where each node in the tree corresponds to a part of the document.

How do you select an element from the DOM?

Selecting elements from the DOM is a crucial part of manipulating web pages with JavaScript. here are some method to manipulate element.

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll
  • children
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript for handling events in a more efficient and flexible way, particularly when dealing with dynamically created elements.

Here are some example:

  • Event Bubbling
  • Event Listener on Parent
  • Event Handling
